Wer ist dabei?

    Mindestens 3 Spieler nötig. Teile den Code, damit alle beitreten können.

    📖 Regelwerk anzeigen

    Grundprinzip

    Jede Runde liest eine wechselnde Moderator:in-Rolle eine ungewöhnliche Frage vor. Alle anderen geben eine möglichst überzeugende Antwort ab (erfunden oder zufällig auch mal die echte). Danach stimmt jede:r ab, welche Antwort die echte ist.

    Punkte

    • Echte Antwort erraten: +3 Punkte
    • Pro Mitspieler, der auf deine erfundene Antwort reinfällt: +2 Punkte
    • Moderator:in bekommt in der Runde keine Punkte

    Schätzen-Karten 🔢

    Türkise Felder (5, 8, 13, 18) lösen statt der normalen Frage eine Schätzen-Karte aus: alle tippen eine Zahl, kein Bluffen, kein Voting. Platz 1 = 3 Punkte, Platz 2 = 2 Punkte, Platz 3 = 1 Punkt, der Rest geht leer aus.

    Aufholjagd 🚀

    Sobald jemand Feld 18 erreicht oder überschreitet, bekommt der/die Letzte einmalig pro Spiel automatisch +5 Bonusfelder. Damit bleibt es bis zum Schluss spannend.

    Spielfeld

    Das Brett hat Felder von 0 bis 26. Wer zuerst Feld 26 erreicht oder überschreitet, gewinnt sofort.

    Ablauf pro Runde

    1. Frage wird gezeigt, alle außer Moderator:in tippen eine Antwort
    2. Abstimmung: welche Antwort ist die echte?
    3. Auflösung mit grün/rot-Feedback
    4. Animiertes Spielbrett zeigt die neuen Positionen
    5. Moderatorrolle wandert weiter, nächste Runde beginnt

    Sonstiges

    Mindestens 3 Spieler nötig. Jede Spielfigur ist pro Raum nur einmal vergebbar. Antworten, die zu ähnlich zur echten Antwort sind, werden beim Absenden automatisch abgelehnt.
